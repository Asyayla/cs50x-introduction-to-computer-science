# DNA Identifier (Python Data Processing)

This Python application identifies a person based on their DNA sequence using Short Tandem Repeats (STRs). It demonstrates proficiency in data manipulation and pattern matching.

## Technical Highlights

- **Data Handling:** Utilized Python's `csv` module to parse and manage relational database files.
- **Pattern Matching:** Implemented an algorithm to find the longest consecutive run of specific DNA subsequences within a larger genomic string.
- **Data Structures:** Used dictionaries and lists to store and compare STR counts across a database of individuals.
- **Complexity:** Designed an efficient nested loop structure to validate matches between the sequence and the database.

## Concepts Demonstrated
- File I/O and CSV parsing
- Command-line argument handling (`sys.argv`)
- String manipulation and slicing
- Algorithmic logic for data comparison
