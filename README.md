# CS50x – Introduction to Computer Science
**Harvard University**

This repository contains selected technical projects and implementations developed during Harvard University's CS50x curriculum. The focus of this collection is to demonstrate fundamental computer science concepts, including algorithmic efficiency, manual memory management, and relational database systems.

---

## Technical Implementations

### C Programming and Memory Management
* **[Speller](./C-Programming-and-Memory/speller):** A high-performance dictionary lookup tool implemented using a Hash Table with separate chaining. Focuses on optimizing search time complexity and efficient memory allocation.
* **[Recover](./C-Programming-and-Memory/recover):** A forensic data recovery application designed to reconstruct deleted JPEG files from raw memory images through byte-level signature analysis.

### Python and Data Processing
* **[DNA Identifier](./Python-and-Data/dna):** A data analysis tool that identifies individuals within a database by matching Short Tandem Repeat (STR) sequences in genomic data.

### SQL and Database Systems
* **[Fiftyville](./SQL-Databases/fiftyville):** An investigative SQL project utilizing complex join operations and nested subqueries to perform data-driven forensics across multiple relational tables.

---

## Core Competencies

The curriculum successfully completed covers the following domains:
* **Systems Programming:** Low-level memory management and pointers in C.
* **Data Structures:** Implementation of Hash Tables, Linked Lists, and Binary Search Trees.
* **Algorithms:** Time and space complexity analysis (Big O notation).
* **Data Engineering:** Relational database design and advanced SQL querying.
* **Web Architecture:** Full-stack development using Flask, SQL, and JavaScript.
