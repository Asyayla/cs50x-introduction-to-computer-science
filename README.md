# Harvard CS50 – Introduction to Computer Science  
## Learning Overview

This repository provides a structured overview of the learning outcomes and academic scope of  
**Harvard University’s CS50: Introduction to Computer Science**.

To fully respect **Harvard’s academic integrity policy**, this repository intentionally  
**does not include solutions** to CS50 problem sets.

---

## Academic Scope

Throughout the course, I completed a rigorous curriculum that covered:

- Computational thinking and problem decomposition (Scratch)
- Low-level programming in **C**  
  - Arrays, algorithms, memory management, pointers, and data structures
- Algorithmic problem solving and efficiency analysis
- Programming in **Python**
- **SQL** and relational database fundamentals
- Web development using **HTML, CSS, and JavaScript**
- Backend development with **Flask**
- Introductory concepts in **Artificial Intelligence**
- A comprehensive **Final Project** integrating multiple course concepts

---

## Technical Skills Developed

- Algorithmic and analytical thinking
- Writing efficient and maintainable code
- Understanding memory-level behavior of programs
- Designing and querying relational databases
- Building basic full-stack web applications
- Debugging, testing, and iterative problem solving
