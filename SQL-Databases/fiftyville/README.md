# Fiftyville (SQL Mystery Solver)

Fiftyville is a data-driven mystery where I acted as a detective to solve a theft using a complex database of phone records, ATM transactions, flight logs, and security footage.

## Technical Highlights

- **Complex Queries:** Developed multi-layered SQL queries utilizing `JOIN`, `IN`, and nested `SELECT` statements to cross-reference suspects.
- **Data Deducting:** Analyzed multiple data sources (bakery security logs, ATM transactions, and flight schedules) to narrow down a large pool of suspects to a single individual.
- **Relational Logic:** Demonstrated a strong understanding of relational database schemas by linking multiple tables through primary and foreign keys.

## Concepts Demonstrated
- Relational Database Management (RDBMS)
- Subqueries and Joins
- Data Analysis and Investigative Logic
- SQL Server / SQLite Syntax
